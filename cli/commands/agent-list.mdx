---
title: 'kapso agent list'
description: 'List agents (local and/or remote)'
---

```bash
kapso agent list [OPTIONS]
```

## Options

| Option            | Description                                     |
| ----------------- | ----------------------------------------------- |
| `--remote, -r`    | Show remote agents from cloud                  |

## Examples

```bash
kapso agent list
kapso agent list --remote
kapso agent list -r
```
