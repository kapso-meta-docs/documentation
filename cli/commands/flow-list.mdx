---
title: 'kapso flow list'
description: 'List flows (local and/or remote)'
---

```bash
kapso flow list [OPTIONS]
```

## Options

| Option            | Description                                     |
| ----------------- | ----------------------------------------------- |
| `--remote, -r`    | Show remote flows from cloud                   |

## Examples

```bash
kapso flow list
kapso flow list --remote
kapso flow list -r
```
