
# message_template_components_update webhook reference

Updated: Nov 14, 2025

This reference describes trigger events and payload contents for the WhatsApp Business Account `message_template_components_update` webhook.

The **message_template_components_update** webhook notifies you of changes to a template’s components.

## Triggers

A template is edited.

## Syntax

```
\{
  "entry": [\
    {\
      "id": "&lt;WHATSAPP_BUSINESS_ACCOUNT_ID&gt;",\
      "time": &lt;WEBHOOK_TRIGGER_TIMESTAMP&gt;,\
      "changes": [\
        {\
          "value": {\
            "message_template_id": &lt;TEMPLATE_ID&gt;,\
            "message_template_name": "&lt;TEMPLATE_NAME&gt;",\
            "message_template_language": "&lt;TEMPLATE_LANGUAGE_AND_LOCALE_CODE&gt;",\
            "message_template_element": "&lt;TEMPLATE_BODY_TEXT&gt;,\
\
            <!-- only included if template has a text header -->\
            "message_template_title": "&lt;TEMPLATE_HEADER_TEXT&gt;",\
\
            <!-- only included if template has a footer -->\
            "message_template_footer": "&lt;TEMPLATE_FOOTER_TEXT&gt;",\
\
            <!-- only included if template has a url or phone number button -->\
            "message_template_buttons": [\
              {\
                "message_template_button_type": "&lt;BUTTON_TYPE&gt;",\
                "message_template_button_text": "&lt;BUTTON_LABEL_TEXT&gt;",\
\
                <!--only included for url buttons -->\
                "message_template_button_url": "&lt;BUTTON_URL&gt;",\
\
                <!--only included for phone number buttons -->\
                "message_template_button_phone_number": "&lt;BUTTON_PHONE_NUMBER&gt;"\
              }\
            ]\
          },\
          "field": "message_template_components_update"\
        }\
      ]\
    }\
  ],
  "object": "whatsapp_business_account"
}
```

## Parameters

| Placeholder | Description | Example value |
| --- | --- | --- |
| `&lt;BUTTON_LABEL_TEXT&gt;`<br />_String_ | Button label text. | `Email support` |
| `&lt;BUTTON_PHONE_NUMBER&gt;`<br />_String_ | Button phone number. | `+15550783881` |
| `&lt;BUTTON_TYPE&gt;`<br />_String_ | [Button type](https://developers.facebook.com/documentation/business-messaging/whatsapp/templates/components#buttons).<br />Values can include:<br />`CATALOG`<br />`COPY_CODE`<br />`EXTENSION`<br />`FLOW`, `MPM`<br />`ORDER_DETAILS`<br />`OTP`<br />`PHONE_NUMBER`<br />`POSTBACK`<br />`REMINDER`<br />`SEND_LOCATION`<br />`SPM`<br />`QUICK_REPLY`<br />`URL`<br />`VOICE_CALL` | `URL` |
| `&lt;BUTTON_URL&gt;`<br />_String_ | Button URL. | `https://www.luckyshrub.com/support` |
| `&lt;TEMPLATE_BODY_TEXT&gt;`<br />_String_ | Template body text. | `Thank you for your order, {{1}}! Your order number is {{2}}. If you have any questions, contact support using the buttons below. Thanks again!` |
| `&lt;TEMPLATE_FOOTER_TEXT&gt;`<br />_String_ | Template footer text. | `Lucky Shrub: the Succulent Specialists!` |
| `&lt;TEMPLATE_HEADER_TEXT&gt;`<br />_String_ | Template header text. | `Your order is confirmed!` |
| `&lt;TEMPLATE_ID&gt;`<br />_Integer_ | Template ID. | `1315502779341834` |
| `&lt;TEMPLATE_LANGUAGE_AND_LOCALE_CODE&gt;`<br />_String_ | Template [language and locale](https://developers.facebook.com/documentation/business-messaging/whatsapp/templates/supported-languages) code. | `en_US` |
| `&lt;TEMPLATE_NAME&gt;`<br />_String_ | Template name. | `order_confirmation` |
| `&lt;WEBHOOK_TRIGGER_TIMESTAMP&gt;`<br />_Integer_ | Unix timestamp indicating when the webhook was triggered. | `1739321024` |
| `&lt;WHATSAPP_BUSINESS_ACCOUNT_ID&gt;`<br />_String_ | WhatsApp Business Account ID. | `102290129340398` |
