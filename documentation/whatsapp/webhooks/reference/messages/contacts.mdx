
# Contacts messages webhook reference

Updated: Oct 22, 2025

This reference describes trigger events and payload contents for the WhatsApp Business Account **messages** webhook for messages containing one or more contacts.

## Triggers

A WhatsApp user sends one or more contacts to a business.
A Whatsapp user sends one or more contacts to a business via a Click to WhatsApp ad.

## Syntax

Note that many contact object properties may be omitted if the WhatsApp user chooses not to share them, or their device prevents them from being shared.

```
\{
  "object": "whatsapp_business_account",
  "entry": [\
    {\
      "id": "&lt;WHATSAPP_BUSINESS_ACCOUNT_ID&gt;",\
      "changes": [\
        {\
          "value": {\
            "messaging_product": "whatsapp",\
            "metadata": {\
              "display_phone_number": "&lt;BUSINESS_DISPLAY_PHONE_NUMBER&gt;",\
              "phone_number_id": "&lt;BUSINESS_PHONE_NUMBER_ID&gt;"\
            },\
            "contacts": [\
              {\
                "profile": {\
                  "name": "&lt;WHATSAPP_USER_PROFILE_NAME&gt;"\
                },\
                "wa_id": "&lt;WHATSAPP_USER_ID&gt;",\
                "identity_key_hash": "&lt;IDENTITY_KEY_HASH&gt;" <!-- only included if identity change check enabled -->\
              }\
            ],\
            "messages": [\
              {\
                "from": "&lt;WHATSAPP_USER_PHONE_NUMBER&gt;",\
                "id": "&lt;WHATSAPP_MESSAGE_ID&gt;",\
                "timestamp": "&lt;WEBHOOK_TRIGGER_TIMESTAMP&gt;",\
                "type": "contacts",\
                "contacts": [\
                  {\
                    "addresses": [\
                      {\
                        "city": "&lt;CONTACT_CITY&gt;",\
                        "country": "&lt;CONTACT_COUNTRY&gt;",\
                        "country_code": "&lt;CONTACT_COUNTRY_CODE&gt;",\
                        "state": "&lt;CONTACT_STATE&gt;",\
                        "street": "&lt;CONTACT_STREET&gt;",\
                        "type": "&lt;CONTACT_ADDRESS_TYPE&gt;",\
                        "zip": "&lt;CONTACT_ZIP&gt;"\
                      }\
                    ],\
                    "birthday": "&lt;CONTACT_BIRTHDAY&gt;",\
                    "emails": [\
                      {\
                        "email": "&lt;CONTACT_EMAIL&gt;",\
                        "type": "&lt;CONTACT_EMAIL_TYPE&gt;"\
                      }\
                    ],\
                    "name": {\
                      "formatted_name": "&lt;CONTACT_FORMATTED_NAME&gt;",\
                      "first_name": "&lt;CONTACT_FIRST_NAME&gt;",\
                      "last_name": "&lt;CONTACT_LAST_NAME&gt;",\
                      "middle_name": "&lt;CONTACT_MIDDLE_NAME&gt;",\
                      "suffix": "&lt;CONTACT_NAME_SUFFIX&gt;",\
                      "prefix": "&lt;CONTACT_NAME_PREFIX&gt;"\
                    },\
                    "org": {\
                      "company": "&lt;CONTACT_ORG_COMPANY&gt;",\
                      "department": "&lt;CONTACT_ORG_DEPARTMENT&gt;",\
                      "title": "&lt;CONTACT_ORG_TITLE&gt;"\
                    },\
                    "phones": [\
                      {\
                        "phone": "&lt;CONTACT_PHONE&gt;",\
                        "wa_id": "&lt;CONTACT_WHATSAPP_PHONE_NUMBER&gt;",\
                        "type": "&lt;CONTACT_PHONE_TYPE&gt;"\
                      }\
                    ],\
                    "urls": [\
                      {\
                        "url": "&lt;CONTACT_URL&gt;",\
                        "type": "&lt;CONTACT_URL_TYPE&gt;"\
                      }\
                    ]\
                  }\
                ],\
\
                <!-- only included if message sent via a Click to WhatsApp ad -->\
                "referral": {\
                  "source_url": "&lt;AD_URL&gt;",\
                  "source_id": "&lt;AD_ID&gt;",\
                  "source_type": "ad",\
                  "body": "&lt;AD_PRIMARY_TEXT&gt;",\
                  "headline": "&lt;AD_HEADLINE&gt;",\
                  "media_type": "&lt;AD_MEDIA_TYPE&gt;",\
                  "image_url": "&lt;AD_IMAGE_URL&gt;",\
                  "video_url": "&lt;AD_VIDEO_URL&gt;",\
                  "thumbnail_url": "&lt;AD_VIDEO_THUMBNAIL&gt;",\
                  "ctwa_clid": "&lt;AD_CLICK_ID&gt;",\
                  "welcome_message": {\
                    "text": "&lt;AD_GREETING_TEXT&gt;"\
                  }\
                }\
              }\
            ]\
          },\
          "field": "messages"\
        }\
      ]\
    }\
  ]
}
```

| `&lt;AD_GREETING_TEXT&gt;`<br />_String_ | Click to WhatsApp ad greeting text. | `Hi there! Let us know how we can help!` |
| `&lt;AD_HEADLINE&gt;`<br />_String_ | Click to WhatsApp ad headline. | `Chat with us` |
| `&lt;AD_ID&gt;`<br />_String_ | Click to WhatsApp ad ID. | `120226305854810726` |
| `&lt;AD_IMAGE_URL&gt;`<br />_String_ | Click to WhatsApp ad image URL. Only included if the ad is an image ad. | `https://scontent.xx.fbcdn.net/v/t45.1...` |
| `&lt;AD_MEDIA_TYPE&gt;`<br />_String_ | Click to WhatsApp ad media type. Values can be:<br />`image` — Indicates an image ad.<br />`video` — Indicates a video ad. | `image` |
| `&lt;AD_PRIMARY_TEXT&gt;`<br />_String_ | Click to WhatsApp ad primary text. | `Summer succulents are here!` |
| `&lt;AD_URL&gt;`<br />_String_ | Click to WhatsApp ad URL. | `https://fb.me/3cr4Wqqkv` |

| `&lt;BUSINESS_PHONE_NUMBER_ID&gt;`<br />_String_ | Business phone number ID. | `106540352242922` |

| `&lt;CONTACT_FIRST_NAME&gt;`<br />_String_ | Contact’s first name. | `Barbara` |
| `&lt;CONTACT_FORMATTED_NAME&gt;`<br />_String_ | Contact’s formatted name. | `Barbara J. Johnson` |
| `&lt;CONTACT_LAST_NAME&gt;`<br />_String_ | Contact’s last name. | `Johnson` |
| `&lt;CONTACT_MIDDLE_NAME&gt;`<br />_String_ | Contact’s middle name. | `Joana` |
| `&lt;CONTACT_NAME_PREFIX&gt;`<br />_String_ | Contact’s name prefix. | `Dr.` |
| `&lt;CONTACT_NAME_SUFFIX&gt;`<br />_String_ | Contact’s name suffix. | `Esq.` |
| `&lt;CONTACT_ORG_COMPANY&gt;`<br />_String_ | Name of the company where the contact works. | `Social Tsunami` |
| `&lt;CONTACT_ORG_DEPARTMENT&gt;`<br />_String_ | Name of the department where the contact works. | `Engineering` |
| `&lt;CONTACT_ORG_TITLE&gt;`<br />_String_ | Contact’s job title. | `Software Engineer` |

| `&lt;CONTACT_STATE&gt;`<br />_String_ | State mentioned in the contact address. | `CA` |

| `&lt;CONTACT_WHATSAPP_PHONE_NUMBER&gt;`<br />_String_ | Contact’s WhatsApp number. | `14125550829` |
| `&lt;CONTACT_ZIP&gt;`<br />_String_ | Zip code in the contact address. | `94025` |
| `&lt;IDENTITY_KEY_HASH&gt;`<br />_String_ | Identity key hash. Only included if you have enabled the [identity change check](https://developers.facebook.com/documentation/business-messaging/whatsapp/business-phone-numbers/phone-numbers) feature. | `DF2lS5v2W6x=` |

| `&lt;WHATSAPP_BUSINESS_ACCOUNT_ID&gt;`<br />_String_ | WhatsApp Business Account ID. | `102290129340398` |
| `&lt;WHATSAPP_MESSAGE_ID&gt;`<br />_String_ | WhatsApp message ID. | `wamid.HBgLMTY1MDM4Nzk0MzkVAgASGBQzQUFERjg0NDEzNDdFODU3MUMxMAA=` |

| `&lt;WHATSAPP_USER_PHONE_NUMBER&gt;`<br />_String_ | WhatsApp user phone number. This is the same value returned by the API as the `input` value when sending a message to a WhatsApp user. Note that a WhatsApp user’s phone number and ID may not always match. | `+16505551234` |
| `&lt;WHATSAPP_USER_PROFILE_NAME&gt;`<br />_String_ | WhatsApp user’s name as it appears in their profile in the WhatsApp client. | `Sheena Nelson` |
